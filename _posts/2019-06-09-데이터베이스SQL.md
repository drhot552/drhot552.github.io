# 데이터베이스
관계형 데이터베이스는 대표적으로 Oracle, Mysql, SQLLite 등의 DBMS가 있습니다.  

회사에서는 보통 Java와 친숙한 Oracle을 사용하고, Javascript언어 기반인 Nodejs는 Mysql 데이터베이스를 사용하고 있습니다.

* Java Server -> Oracle

* JavaScript -> Mysql
